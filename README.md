# Speciale
Dette repository indeholder data og R-kode anvendt i mit speciale i statskundskab, december 2025.

# Indhold
`clean_excel_sheet_28okt.xlsx`: Datasæt anvendt i analyserne
`ny_population.xlsx`: Populationsdata fra Danmarks Statistik anvendt til sammenligning af stikprøve med population
`Kodning af specialedata/`: R-script til datahåndtering og analyse
`Power-analyse/`: R-script til datahåndtering og analyse
